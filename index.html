JSinject Documentation

Injector

JSinject features an injector for JavaScript code. It is only recommended for experienced developers, but if you have a JavaScript file (currently online only) you can fetch it with this simple code:
fetch("The URL").then((res) => res.text()).then((js) => eval(js));
Replace "The URL" with the .js. Usually, with GitHub, the URL should be something like https://raw.githubusercontent.com/us....file.js. If your file is like this, congratulations!

Troubleshooting

If you encountered an error while trying to run your script, here are some common problems and fixes:

  1. Redirects/New windows not allowed by github.io or disnos9.github.io.
  2. Your code is not formatted correctly. Learn more about using Injector.
  3. We were unable to open an about:blank window.

This will only appear if the about:blank window fails to appear.
